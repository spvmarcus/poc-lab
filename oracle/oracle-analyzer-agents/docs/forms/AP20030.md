## Regras de validação e consistência (visão geral)

- valida cartões por datas (comparação com data_admissao, data_demissao, mês fechado)
- valida presença de registro em RDH_FUNCIONARIO_MES
- valida horas (entrada < almoço < retorno < saída)
- valida flags (periodico/falta, não altera autorizados, não permite inclusão de sub.global)
- valida estado de aprovação (aprovacao_chefia, aprovacao_rh)

## Dependências indiretas

- usa as procedures e funções: calcula_horas_periodico, calculo_de_horas, consiste_lancamentos, cons_ativ, cons_ativ_02_05_09, ver_atividade, ver_projeto, verifica_horario, etc.
